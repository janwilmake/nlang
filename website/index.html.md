### 2. How It Works (3-step horizontal cards)

1. **Name it** — `index.html.md` → the last extension picks the executor
2. **Write it** — Markdown becomes an LLM prompt, JS/TS runs in Node
3. **Build it** — `nlang build` outputs `dist/index.html`
